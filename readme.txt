== Description ==

*simple post series* allows you to insert series short code for displaying a bunch of posts in the same series.

== Installation ==

1. Uzip the `post-series.zip` folder.
2. Upload the `post-series` folder to your `/wp-content/plugins` directory.
3. In your WordPress dashboard, head over to the *Plugins* section.
4. Activate *Post Series*.
